HOW TO ADD PRODUCT IMAGES
==========================
1. Find the product on Amazon
2. Right-click the product image → Save Image As
3. Rename it to match the filename shown in the image placeholder on your page
4. Upload it to this /images/ folder in GitHub
5. The image will automatically appear on your site
